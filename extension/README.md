# Keyword Master 자막 도우미 (크롬 확장)

유튜브는 서버(데이터센터 IP)의 자막 요청을 봇으로 차단하기 때문에, 자막 추출은 사용자의
브라우저에서 수행합니다. 이 확장은 사이트에서 영상 분석 시 자막을 대신 추출해 채워 넣습니다.
확장이 설치되지 않은 브라우저에서는 자막만 "스크립트 없음"으로 표시되고 나머지 기능은 정상 동작합니다.

## 설치 (개발자 모드)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 `extension` 폴더 선택

## 서비스 도메인 추가

기본으로 `localhost`에서만 동작합니다. 사이트를 실제 도메인에 배포하면
`manifest.json`의 `content_scripts.matches`에 도메인을 추가하고 확장을 새로고침하세요.

```json
"matches": ["http://localhost/*", "http://127.0.0.1/*", "https://내도메인.com/*"]
```

## 동작 방식

1. 사이트에서 영상 분석 시 자막이 비어 있으면 페이지가 `YKM_TRANSCRIPT_REQUEST` 신호를 보냄
2. 확장이 사용자 브라우저의 IP로 유튜브 시청 페이지에서 자막 트랙을 추출
   (게시자 수동 자막 우선, 없으면 자동 생성 자막 / 한국어 우선)
3. 추출된 자막을 사이트에 돌려주면 사이트가 DB에 저장하고 화면에 표시
