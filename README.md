# YouTube Keyword Master

유튜브 카테고리별 인기 영상의 성장 지표를 확인하고, 관심 영상을 분석해 Excel로 저장하는 대시보드 MVP입니다.

## 실행

```bash
npm install
npm run dev
```

프로덕션 빌드는 `npm run build`로 생성합니다.

## 구현 범위

- 전체 급상승 + 9개 카테고리 인기 차트(카테고리당 최대 200개), 검색, 정렬
- 조회수·좋아요의 직전 수집 대비 증가량
- 세로 영상을 Shorts, 가로 영상을 일반 영상으로 분류
- 영상 클릭 분석과 결과 누적
- 채널, 제목, 링크, 길이, 등록일, 좋아요, 댓글 수, 인기 댓글 TOP 3, 해시태그, 스크립트 표시
- 분석 결과를 실제 `.xlsx` 파일로 저장
- 수집 주기, API 상태, 데이터 보관, 알림 설정 UI
- 데스크톱/모바일 반응형 레이아웃

Supabase 환경변수가 설정되면 프론트엔드는 임시 데이터를 사용하지 않고 `youtube_keyword_master_` 접두사가 붙은 테이블과 뷰만 조회합니다. Supabase가 설정되지 않은 로컬 데모 환경에서만 샘플 데이터로 동작합니다.

## Supabase 구성

원격 배포 절차는 [supabase/README.md](supabase/README.md)를 참고하세요.

YouTube API 키는 브라우저 코드에 포함하지 않고 Edge Function 비밀값으로만 관리합니다. 스크립트(자막)는 YouTube Data API가 제공하지 않으므로 분석 시 시청 페이지를 크롤링해 추출합니다(게시자 수동 자막 우선, 없으면 자동 생성 자막). 자막이 없거나 유튜브가 서버 접근을 차단한 경우 `스크립트 없음`으로 표시합니다.
